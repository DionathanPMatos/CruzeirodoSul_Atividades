# Plataforma Social – Projeto Acadêmico

Este repositório contém o projeto desenvolvido ao longo das entregas da disciplina, evoluindo de uma estrutura HTML estática para uma aplicação web com CSS responsivo, JavaScript com SPA básica, acessibilidade e deploy.

## 1. Visão Geral

- **Tema:** Organização Social Esperança
- **Tecnologias utilizadas:** HTML5, CSS3, JavaScript (ES6), Git/GitHub
- **Entrega final:** Estrutura semântica, design system, SPA, acessibilidade (WCAG 2.1 AA básica) e otimização para produção.
